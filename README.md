# 📱 Proyecto Arkana Admin

Aplicación móvil desarrollada en **React Native** para la administración del juego de cartas NFC. El proyecto implementa interfaces modernas, accesibles y responsivas orientadas a la gestión de usuarios, tarjetas NFC y estadísticas del sistema.

---

## 🚀 Características

### 🔐 Autenticación

* Inicio de sesión para administradores.
* Validación de credenciales.
* Navegación entre pantallas.

### 👤 Gestión de Usuarios

* Registro de nuevos usuarios.
* Formularios intuitivos y accesibles.
* Preparado para integración con API REST.

### 🃏 Administración de Tarjetas NFC

* Visualización de tarjetas registradas.
* Operaciones CRUD (Crear, Consultar, Actualizar y Eliminar).
* Interfaz optimizada para dispositivos móviles.

### 📊 Dashboard Administrativo

* Estadísticas generales.
* Usuarios activos.
* Tarjetas NFC registradas.
* Accesos rápidos a los diferentes módulos.

### ⚙️ Perfil y Accesibilidad

* Información del usuario activo.
* Opciones de accesibilidad.
* Diseño adaptable para distintos tamaños de pantalla.

## ▶️ Instalación

Clonar el repositorio:

```bash
git clone https://github.com/LyenZo/arkana_admin_movil.git
```

Ingresar al proyecto:

```bash
cd arkana-mobile
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npx expo start
```
